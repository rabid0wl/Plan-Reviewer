## Permit Process — Buena Park

### Application Processing (from ADU ordinance)

- **Approval type**: Ministerial — no discretionary review, no public hearing
- **Timeline**: 60 days from receipt of complete application
- **Extension**: City and applicant may agree to additional time with written request
- **Denial**: Within 60 days, return full written comments listing deficient items with description of how to correct
- **Garage replacement**: Demolition permit required, reviewed concurrently with ADU application

### Plan Submittal Requirements

- **3 sets of plans** + **2 sets of structural calculations** (if applicable)
- First page must be **wet stamped** by architect or engineer
- Subsequent pages may be electronic
- Structural calculations must be **original, wet stamped, and signed**
- **Residential plans** CAN be prepared by anyone (no licensed professional required for residential)
- Plans submitted and resubmitted **in-person only** — no courier, electronic, or mail
- Initial submittal should NOT be clouded (clouds indicate a revision to an issued permit)
- Plans must be legible and suitable for printing

### Code Cycle Requirement

Plans must reference **2025 Edition** California Building Standards Code (effective January 1, 2026). Incorrect code cycle information will NOT be accepted.

### Required Plan Sheets for ADU

Based on ordinance requirements and Room Addition guidelines:

1. **Site/Plot Plan** — Lot dimensions, side yards, distances between buildings (proposed and existing), directional orientation
2. **Foundation Plan** — Fully dimensioned with applicable details
3. **Floor Plan** — Fully dimensioned and labeled, including adjacent existing room dimensions
4. **Framing Sections** — With appropriate labels, notes, and structural details
5. **Exterior Elevations** — Scaled drawings showing compliance with State Energy Conservation Standards
6. **Demolition Plan** — If applicable (e.g., garage demolition for ADU)

### Additional Requirements

- Engineering Services grading certification waiver
- Fire sprinkler evaluation by Building Division
- School impact fees if exceeding 499 sq ft
- Possible Orange County Sanitation fees
- Elevation permit from City Engineer (required before building permit — see `building-codes.md`)
