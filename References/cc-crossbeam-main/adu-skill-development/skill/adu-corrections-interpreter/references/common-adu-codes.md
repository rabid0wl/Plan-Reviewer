# Common California ADU Building Codes

Quick-reference for codes frequently cited in ADU corrections letters. Use this to quickly identify what a code reference covers before doing deeper web research.

## California Residential Code (CRC)

| Section | Topic | Common Correction |
|---------|-------|-------------------|
| **R302.1** | Exterior wall location / fire separation distance | Projections (eaves, patio covers) within 5' of property line need 1-hour fire rating on underside. Options: 5/8" Type X gypsum, FRT wood, or heavy timber. |
| **R302.1 Table R302.1(1)** | Fire separation distance table | Specifies requirements at <2', 2-5', and 5'+ from property line for walls, projections, and openings. |
| **R301.2** | Climatic and geographic design criteria | Wind speed, seismic design category, snow load, frost depth for the jurisdiction. |
| **R311** | Means of egress | Exit doors, hallways, landings, stairways. Landing size and configuration. |
| **R403** | Footings | Foundation depth, width, reinforcement. |
| **R602** | Wood wall framing | Stud size, spacing, headers, bracing. |
| **R802** | Wood roof framing | Rafter size, span tables, ridge board, ceiling joists. |

## California Building Code (CBC)

| Section | Topic | Common Correction |
|---------|-------|-------------------|
| **Chapter 7A** | Materials and construction for wildfire exposure | Applies in WUI (Wildland-Urban Interface) zones. Roofing, eaves, vents, exterior walls. |
| **Table 601** | Fire-resistance rating requirements | Type of construction determines required fire ratings. |
| **Section 1613** | Earthquake loads | Seismic design requirements referencing ASCE 7. |

## ASCE 7-16 (Minimum Design Loads)

| Section | Topic | Common Correction |
|---------|-------|-------------------|
| **Section 30.11** | C&C wind loads for attached canopies | New in ASCE 7-16. Wind load calcs for covered patios/canopies attached to buildings. Uses Table 30.11-1 and Figures 30.11-1A/1B. |
| **Section 12** | Seismic design requirements | Seismic base shear, lateral force distribution. |
| **Section 26-30** | Wind loads (general) | MWFRS and C&C wind load provisions. |
| **Chapter 2** | Load combinations | How to combine dead, live, wind, seismic loads. |

## California Business & Professions Code

| Section | Topic | Common Correction |
|---------|-------|-------------------|
| **5536.1** | Architect stamp/signature requirements | All drawings must be stamped and signed by the architect of record. Wet signature required on all sheets. |
| **5802** | Professional engineer stamp requirements | Engineering documents must bear the engineer's stamp. |
| **6735** | Civil engineer stamp requirements | Civil engineering documents must be stamped by a licensed civil engineer. |

## California Plumbing Code (CPC)

| Section | Topic | Common Correction |
|---------|-------|-------------------|
| **Table 702.1** | DFU (Drainage Fixture Unit) values | Used to size drain/waste/vent piping based on number and type of fixtures. |
| **Section 610** | Water supply sizing | Sizing water supply lines based on fixture count and demand. |

## City-Level Requirements (Varies by Jurisdiction)

Common city-specific items that appear in corrections letters:

| Requirement | What to Search For |
|-------------|-------------------|
| **ADU sewer connection** | Search city website for "ADU sewer connection standard detail" â€” many cities publish a PDF with the exact connection drawing. |
| **Municipal code on cover sheet** | The city wants their local municipal code listed as a governing code because it contains local amendments to state codes. |
| **Grading/drainage slopes** | Typically 5% at landscape, 2% at hardscape away from foundation. Check the city's grading chapter. |
| **"For Reference Only" on grading sheets** | Standard practice when grading is covered under building permit, not a separate grading permit. |
| **ADU development standards** | Cities have local ADU ordinances (setbacks, height, FAR, parking). Usually in the zoning title of their municipal code. |
