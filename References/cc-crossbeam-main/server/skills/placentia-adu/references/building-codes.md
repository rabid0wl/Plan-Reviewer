## Building Codes — Placentia

### Adopted Code Editions (Title 20)

| Chapter | Code | Edition | Base Code |
|---------|------|---------|-----------|
| 20.04 | California Building Code (CBC) | **2022** | 2021 IBC |
| 20.08 | California Residential Code (CRC) | **2022** | 2021 IRC |
| 20.12 | California Plumbing Code (CPC) | **2022** | 2021 UPC |
| 20.16 | California Electrical Code (CEC) | **2022** | 2020 NEC |
| 20.20 | California Mechanical Code (CMC) | **2022** | 2021 UMC |
| 20.40 | Grading and Excavating | Local | City grading code |
| 20.46 | California Energy Code (CEnC) | **2022** | Title 24 Part 6 |
| 20.50 | California Existing Building Code (CEBC) | **2022** | 2021 IEBC |

### IMPORTANT: 2025 Code Cycle

**Plans accepted for review on or after January 1, 2026 must comply with the 2025 California Building Code.** This applies to all plan check submittals including ADU projects.

### Local Amendments

- **CBC amendments:** Included as attachment to Chapter 20.04
- **CRC amendments:** Included as attachment to Chapter 20.08
- **Fire Code amendments:** Included as attachment to Chapter 18.04 (Section 18.04.050)
- CBC adopted including **all appendices** by reference as the Building Code of the City of Placentia
- Enforced by the city building official or designee

### Grading Code (Chapter 20.40)

- Placentia has its **own grading code** (NOT the Orange County grading code used by some OC cities)
- No grading within the city without a permit
- Grading permits approved and issued by the **city engineer**
- Contact: Public Works Engineering at (714) 993-8131
