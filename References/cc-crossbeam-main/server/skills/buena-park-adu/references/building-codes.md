**IMPORTANT — Code Cycle**: Plans submitted after **January 1, 2026** must reference the **2025 Edition** of the California Building Standards Code. The city explicitly states incorrect code cycle information will NOT be accepted. However, the city's own municipal code has NOT yet been updated to formally adopt the 2025 editions as of July 2025.

### Local Building Code Amendments (§ 15.56.020)

The city has adopted the 2022 CBC with local amendments. Key amendments:

1. **Modified definitions** — Local usage modifications
2. **Address identification requirements** — Address numbers and visibility standards
3. **Sprinkler and standpipe systems** — Design of hydraulically calculated fire sprinkler systems shall not exceed **90% of water supply capacity** (exception: when static pressure exceeds 100 psi)
4. **Automatic fire-extinguishing systems** — Required in all occupancies when total building area exceeds **5,000 sq ft** or is **more than 2 stories** in height

**QCode URL for amendments**: https://qcode.us/codes/buenapark/view.php?frames=on&topic=15-15_56-15_56_020

### Elevation Permits (Ch. 15.20)

Required from City Engineer **before** construction of any building/structure that requires a building permit. Establishes finished floor elevations and drainage patterns.
